退换货政策

通过分期商城购买的商品，在符合退换货政策的条件下，收到商品之日起7日内可以申请退换货，15日内可以换货，您可在线提交申请办理退换货事宜或致电商户客服热线申请退换货。具体退换货标准及操作流程如下：

你可以通过微信客服“中银消费钱包分期商城客服”完成退换货申请：

一、退换货标准

(一)7天无理由退货：

自您实际收到商品之日起7日内，在商品保持完好，且您愿意自行承担退货运费的情况下，可申请无理由退货。我们承诺销售商品均为全新商品，为保护消费者利益，以下商品不适用于7天无理由退换货：

1.个人定制类商品；

2.鲜活易腐类商品；

3.在线下载或者已拆封的音像制品，计算机软件等数字化商品；

4.交付的报纸期刊类商品；

5.其他根据商品性质并在购买时经您确认不宜退货的商品；

(二)商品质量问题的退换货：

因以下原因需要退、换货的，可以在您实际收到商品之日起7日内申请退货，在15日内申请换货。退换货产生的运费均由商户承担。

1.国家法律所规定的功能性故障或商品质量问题：经由生产厂家指定或特约售后服务中心检测确认，并出具检测报告或经第三方卖家售后确认属于商品质量问题。

2.商品实物与网站描述不符或商品原装配件缺失：

3.商品在运输过程中造成的损坏、漏液、破碎、性能故障，经商户核查情况属实。

(三)无法办理退换货的情形：

1.任何非中银消费钱包分期商城出售的商品（序列号不符）；

2.已超过超过三包保修期的商品；

3.商品已被损坏，如未经授权的维修、误用、碰撞、疏忽、滥用、进液、事故、改动、不正确的安装所造成的商品质量问题，或撕毁、涂改标贴、机器序号、防伪标记；

4.其他依据国家相关法律法规不应办理退换货的。

二、退换货流程

在微信“中银消费钱包分期商城客服”中发送“退货/换货”指令；

按照提示输入订单号；

按照提示输入金额和购买人手机号末4位进行确认；

至此，您一旦提交申请，该申请单就进入了商城的售后服务处理流程。客服会根据您填写的申请单情况给予回应。当您的申请单审核通过时，您需要邮寄商品至审核回复后的地址。在邮寄时，请您将所有退货、返修的物品都放入一个邮包中一起寄送。

三、温馨提示

(一)礼包或套装中的商品不可以部分退货，有绑定赠品的主商品要将赠品一同退换货。

(二)图片及信息仅供参考，不属质量问题。因拍摄灯光及不同显示器色差等问题可能造成商品图片与实物有色差，请以实物为准。
