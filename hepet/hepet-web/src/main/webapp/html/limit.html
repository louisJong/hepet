• 贷款审核时间大约多久？

    正常审核时长在20分钟至1小时不等，审核结果会有短信通知。

  • 申请为什么会被拒？

    审批不通过很可能是因为综合评分不足，我们会根据您的实际拒绝原因在APP端告诉您是否需要重新申请。

  • 审核会打公司或者联系人电话吗？

    不会，但为确保您能够成功申请到额度，请务必填写真实信息。

  • 为什么提示我消费借款申请资格验证失败？

    可能有两种原因导致这种情况：
    1：您有正在审批中的贷款，请耐心等待，勿重复申请；
    2：您已有我司其他产品，可以先结清其他产品再进行申请。

  • 为什么我人脸识别反映很慢，一直不过？

    若您的脸不在镜头内、不清晰、当前灯光昏暗或佩戴眼镜造成反光，都可能导致刷脸失败。建议在光线明亮的室内进行录制，并摘掉眼镜。如果以上问题都不存在，可能是网络延时，建议稍后再试。

  • 为什么扫描身份证反映很慢，一直不过？

    网络问题、灯光昏暗、身份证反光都可能导致扫描较慢，若10秒内扫描不成功需手动点击拍摄按键，请注意关注扫描过程中页面的变化。
