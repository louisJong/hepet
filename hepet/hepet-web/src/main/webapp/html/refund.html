• 中银消费钱包的还款方式是怎样的？

    每月1至31号为一个记账周期，次月2号出账，账单将包含您上月的消费金额以及分期应还金额，届时APP会显示您最新的账单信息，并可查询您的所有用款记录，建议每月2号登录APP查查账单。 同时我们每月2号和13号也会短信告知您当期应还的费用，并会详细告知其中有多少金额可以申请账单分期，请关注我们的短信。

  • 忘记还款怎么办？

    若您在还款日忘记还款则应在还款日后3个自然日之内完成还款，此时不进入催收，亦不上报人民银行征信。此3天计收滞纳金（贷款金额的0.05%/天）。若超过3个自然日仍未还款则纳入恶意逾期，纳入人行征信系统，同时余下分期自动结清为当月账单。

  • 扣款失败了，显示我的账户有问题，怎么回事？

    快去确认一下您绑定的借记卡中余额是否充足，账户状态是否正常，是否有挂失、冻结等情况。如果余额充足、状态正常，系统第二天会再次扣款的，扣款结果会有短信通知。

  • 如何计算分期手续费？

    中银消费钱包3/6/9/12期的分期费率分别是当笔用款金额的1.98%、1.74%、1.65%、1.6%；现金分期、账单分期及订单分期费率相同。
